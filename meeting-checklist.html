현황 분석

아임웹에서 독립몰로, 왜?

현재: 아임웹 호스팅

아임웹은 임대 아파트와 같습니다.
입주(가입)하면 바로 쇼핑몰을 운영할 수 있지만,
벽을 허물거나 구조를 바꿀 수는 없습니다.

✗ 기능 확장 거의 불가 — 커스텀 개발 제약이 큼
✗ 네이버/구글 검색에 잘 안 잡힘 — 검색 설정을 직접 못 바꿈
✗ 데이터(회원/주문)를 직접 소유하지 못함
✗ 사이트 속도를 직접 빠르게 만들 수 없음
전환 후: 독립 쇼핑몰 (자가 소유)

독립몰은 내 건물을 짓는 것입니다.
원하는 구조로 자유롭게 설계하고,
모든 데이터를 직접 소유합니다.

✓ 원하는 기능 무제한 추가/커스터마이징 가능
✓ 검색 노출 완전 제어 — 네이버/구글 유입 극대화
✓ 회원/주문/매출 데이터 100% 소유
✓ 서버 관리 걱정 없이, 접속자가 몰려도 자동으로 버틸 수 있는 구조

기존 회원 정보, 상품, 주문 내역 모두 안전하게 옮겨드립니다. 고객분들은 변화를 거의 못 느끼십니다. 저희가 처리합니다.

데이터 이관
핵심

회원 · 상품 · 주문 데이터, 통째로 옮겨드립니다

✓ 저희가 해드리는 것
✓ 회원/상품/주문 데이터: 아임웹에서 자동 추출 → 새 시스템으로 이관
✓ 상품 데이터: 약 36개+ 이미 파악 완료, 8개 카테고리
✓ 상품 사진: 자동 다운로드 → 새 이미지 저장소로 이관 (해지 전 먼저 수집)
✓ 비밀번호: 이전 불가 → 임시 비밀번호 발급 + 재설정 안내까지 세팅
✓ 포인트/등급/위시리스트: 자동 추출하여 이관
📋 대표님께 필요한 것
☐ 아임웹 관리자 로그인 정보 공유
☐ 추가 이관 데이터 확인 (포인트, 쿠폰, 구매평 등 필요한지)
☐ 아임웹 해지 시점은 저희가 안내 (오픈 후 1~2주 유지 권장)
☐ 회원 수 / 상품 수 (이관 규모 파악)
☐ 비밀번호 재설정 안내 방법 (카카오 알림톡 vs 이메일 vs SMS)

아임웹의 모든 데이터(회원/상품/주문/이미지)를 자동으로 추출하는 도구를 이미 준비했습니다. 로그인 정보만 공유해주시면 저희가 전부 처리합니다.

결제 & 도메인
핵심

카드결제 이전 + 사이트 주소 이관, 전부 처리합니다

✓ 저희가 해드리는 것
✓ 현재 카드결제 회사 파악 → 최적 이관 경로 설계
✓ 새 가맹점 등록 즉시 접수 (심사 서류 준비도 저희가)
✓ 도메인 이전: 인증번호 발급 요청 → 주소 이전 일괄 진행
✓ 보안 인증서: 새로 무료 발급 (자물쇠 마크)
✓ 네이버 메일 사용 중 → 이메일 영향 없음 ✓
✓ 기존 주소 자동 연결 설정 (유입 보존)
📋 대표님께 필요한 것
☐ 현재 카드결제 회사 이름 (어떤 회사를 쓰시는지만)
☐ 사용 중인 결제 수단 (카드/무통장/간편결제)
☐ 사업자등록증, 통신판매업 신고증
☐ 도메인 어디서 구매했는지 (아임웹 vs 가비아/후이즈 등)

참고: 카드결제 심사 1~2주 소요 — 14일 일정의 최대 병목. 미팅 직후 심사 접수가 필요합니다. 저희가 바로 접수합니다.

배송 · 디자인 · 알림 · 관리자 페이지, 모두 구축합니다

✓ 배송: 우체국 택배 연동 (당일 빠른 배송, 군부대 가능, 묶음배송 가능)
