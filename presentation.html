SECTION 1 · 프로젝트 이해

SaaS 빌더 기반 쇼핑몰의 한계

🔒

기능 확장의 한계

카페24, 아임웹, 고도몰 등 SaaS 플랫폼은 기본 기능은 빠르게 제공하지만, 비즈니스 고유의 커스텀 기능이나 외부 시스템 연동에 제약이 큽니다.

💰

지속적인 호스팅 비용

월 이용료, 트래픽 초과 과금, 유료 앱 추가 비용 등이 매월 발생하며, 장기적으로 자체 구축 대비 비용 효율이 떨어집니다.

📊

데이터 종속

고객 데이터와 상품 데이터가 플랫폼에 종속되어 있어, 자유로운 데이터 활용과 분석, 이관이 어렵습니다.

🎯

클라우드 독립몰 전환 기회

서버리스 클라우드(AWS) 기반으로 전환하면 서버 관리 부담 없이 완전한 기술 통제권을 확보하고, 원하는 기능을 자유롭게 구현할 수 있습니다.

SECTION 1 · 프로젝트 이해

프로젝트 목표

1

호스팅 서버 없이 운영 가능한 쇼핑몰

AWS 서버리스 아키텍처로 서버 관리 없이 자동 확장되는 쇼핑몰을 구축합니다. 트래픽에 따라 비용이 유동적으로 조절됩니다.

2

기본 쇼핑몰 기능 완벽 구현

회원가입/로그인, 상품 등록·검색·상세, 장바구니, 결제(토스페이먼츠), 주문·배송 관리까지 핵심 커머스 기능을 모두 구현합니다.

3

기존 고객DB & 상품DB 이관

기존 쇼핑몰의 회원 정보와 상품 데이터를 신규 시스템(MySQL on AWS RDS)으로 안전하게 마이그레이션합니다.

4

PC/모바일 반응형 UI

PC, 태블릿, 모바일 모든 기기에서 최적화된 쇼핑 경험을 제공하는 반응형 웹을 구축합니다.

SECTION 3 · 핵심 기능

쇼핑몰 핵심 기능

👤

회원가입 / 로그인

이메일 회원가입, 소셜 로그인(카카오, 네이버), JWT 기반 인증. 마이페이지에서 주문 내역 및 개인정보 관리.

📦

상품 관리

상품 목록, 카테고리 필터, 검색, 상세 페이지. 관리자 페이지에서 상품 등록/수정/삭제 및 재고 관리.

🛒

장바구니 & 결제

장바구니 담기/수량 변경/삭제, 토스페이먼츠 PG 연동으로 카드·계좌이체·가상계좌 결제를 지원합니다.

🚚

주문 / 배송 관리

주문 상태 자동 업데이트, 송장번호 입력 및 배송 추적, 반품/교환 처리 플로우를 구현합니다.

SECTION 3 · 핵심 기능

데이터 이관 전략

1

데이터 추출

기존 쇼핑몰 API 또는 Export 기능으로 회원 정보, 상품 정보, 주문 이력 등 핵심 데이터를 추출합니다.

2

스키마 매핑 설계

기존 데이터 구조 → 신규 MySQL 스키마로의 매핑 테이블을 설계합니다. 필드명, 데이터 타입, 관계를 정의합니다.

3

무결성 검증

중복 데이터 체크, 데이터 타입 검증, 누락 항목 확인으로 이관 품질을 보장합니다. 테스트 이관을 먼저 실행합니다.

4

본 이관 실행 & 검수

검증 완료 후 본 이관을 실행합니다. 기존 회원에게는 비밀번호 재설정 안내를 발송하고, 이관 후 데이터 정합성을 최종 검수합니다.

회원 목록 조회, 검색, 상태 관리. 주문 이력 및 활동 내역을 확인할 수 있습니다.
