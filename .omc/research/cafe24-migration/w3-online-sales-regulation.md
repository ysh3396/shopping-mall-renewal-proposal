# 전자담배 온라인 판매 관련 규제 상세

> 조사일: 2026-03-29
> 작성자: worker-3

---

## 1. 연령인증(19+) 요구사항 및 기술적 구현 방법

### 법적 근거
- **청소년보호법 제28조제1항**: 누구든지 청소년을 대상으로 담배를 판매할 수 없으며, 자동기계장치·무인판매장치·**통신장치(온라인 포함)**를 통한 판매도 포함
- **청소년보호법 제28조제4항**: 담배 판매자는 상대방의 **나이 및 본인 여부를 반드시 확인**해야 함
- **국민건강증진법 시행규칙 제5조의2**: 담배자동판매기에 성인인증장치 부착 의무

### 인증 방법
| 인증 수단 | 설명 | 온라인 적용 가능 여부 |
|-----------|------|---------------------|
| 주민등록증 인식 | 실물 신분증 스캔/OCR | 오프라인 전용 |
| 운전면허증 인식 | 실물 신분증 스캔/OCR | 오프라인 전용 |
| 휴대폰 본인인증 | 통신사 기반 실명/연령 확인 | 온라인 가능 |
| 아이핀(i-PIN) | 주민번호 대체 인증 | 온라인 가능 |
| 간편인증(PASS, 네이버, 토스 등) | 민간 인증서 기반 | 온라인 가능 |
| 신용카드/직불카드 인증 | 금융거래 기반 성인 확인 | 온라인 가능 |

### 벌칙
| 위반 사항 | 처벌 |
|-----------|------|
| 청소년에게 담배 판매 | **2년 이하 징역 또는 2,000만원 이하 벌금** |
| 성인인증장치 미부착 | 1차 75만원, 2차 150만원, 3차 이상 300만원 과태료 |
| 담배자판기 부당 설치 | 1차 170만원, 2차 330만원, 3차 이상 500만원 과태료 |

### 기술적 구현의 어려움
- 단순 생년월일 입력으로는 법적 요건 미충족 → 반드시 **본인인증 API 연동** 필요
- 회원가입 시 1회 인증으로는 부족, **구매 시점마다 재인증** 또는 **인증 유효기간 관리** 필요
- 비회원 구매 시에도 성인인증 강제 적용 필요
- 타인 명의 도용 방지를 위한 추가 검증 수단 필요 (안면인식 등)

## 2. 광고 제한

### 법적 근거
- **국민건강증진법 제9조의4**: 담배에 관한 광고 금지 또는 제한
- **2026년 4월 24일 담배사업법 개정**: 합성니코틴 포함 전자담배도 동일 규제 적용

### 허용되는 광고 (매우 제한적)
| 허용 채널 | 조건 |
|-----------|------|
| 소매점 내부 | 표시판·스티커·포스터 전시/부착만 가능 |
| 잡지 광고 | 연간 10회 이내, 1회당 2쪽 이내 |
| 행사 후원 | 여성·청소년 미대상 행사만 가능 |
| 국제 운송 수단 | 국제선 항공기·여객선에서만 허용 |

### 전면 금지 항목
- **TV 광고**: 전면 금지
- **온라인 광고**: 전면 금지 (웹사이트, 배너, 검색광고 모두 포함)
- **SNS 마케팅**: 전면 금지 (인스타그램, 유튜브, 틱톡 등)
- **네이버 광고**: 전자담배, 궐련, 액상, 액세서리 포함 모든 종류 **엄격히 금지**
- **네이버쇼핑**: 제품 등록은 가능하나 광고는 불가

### 광고 내용 제한
- 비흡연자에게 직·간접적으로 흡연을 권장·유도하는 내용 금지
- **여성 또는 청소년 인물 묘사 금지**
- 검증되지 않은 건강 관련 내용 표시 금지
- 흡연경고문구의 내용·취지에 반하는 내용 금지
- 가향 물질(수박향, 망고향 등) 홍보 불허

### 광고물 필수 포함 사항
- 폐암 등 질병 위험성 경고
- 발암성물질 명시
- 금연상담전화(1544-9030) 표기

### 위반 시 벌칙
- **1년 이하 징역 또는 1,000만원 이하 벌금**
- 판촉금 제공 위반: 500만원 이하 벌금

## 3. 성분표시 의무사항

### 현행 규정
- **국민건강증진법 시행령 제16조의2**: 담배 포장지 및 광고에 경고문구 표시 의무
- **국민건강증진법 시행규칙 제6조의3**: 경고문구 세부 기준

### 표시 의무 항목
| 항목 | 내용 |
|------|------|
| 경고문구 | 흡연의 건강 위험성 경고 |
| 경고그림 | 담뱃갑 포장지에 시각적 경고 |
| 성분 표기 | 니코틴 함량 및 유해성분 표시 |
| 제조/수입 정보 | 제조자, 수입자 정보 |

### 2026년 4월 24일 이후 강화 사항
- 합성니코틴 액상형 전자담배에도 **경고문구·경고그림 표기 의무화**
- 담배 성분 표기 의무 확대
- **담배유해성관리법**에 따른 유해성분 검사 실시
- 법 시행 전후 제조 물량 구분을 위한 추가 식별조치

### 현실적 문제
- 현재 전자담배 업계에서 니코틴 표시함량과 실제함량 차이가 **최소 3.1% ~ 최대 29.6%**로 나타남
- 기존에는 합성니코틴 제품에 대한 성분/유해성 검증 의무가 없었으나, 개정법 시행 후 강화될 예정
- 온라인 쇼핑몰에서는 상품 상세페이지에도 법정 경고문구 및 성분 정보를 표시해야 함

## 4. 결제 제한

### PG사(결제대행사) 심사 제한

#### 전자담배 업종의 PG 가입 현황
- **포트원(구 아임포트) 기준**: 전자담배는 "2026.04.10~ 온라인 판매 전면 금지 사전 권고 진행 중"으로 분류
- 전자담배 액상·키트 판매는 **고위험업종**으로 분류되어 PG사 입점 사실상 불가
- 카드사 및 PG사 등록 불가 기준은 사회적 이슈 등에 따라 **수시로 변경**

#### 간편결제 제한
| 결제 수단 | 전자담배 취급 가능 여부 |
|-----------|----------------------|
| 네이버페이 | **취급 불가** (전자담배액체 금지 품목) |
| 카카오페이 | 제한적 (성인인증 별도 검토 필요) |
| 토스페이 | PG사 정책에 따라 제한 |
| 일반 신용카드 | 카드사별 개별 심사, 거절 가능성 높음 |

#### PG사 거절 사유
- 결제사고가 빈번한 업종으로 분류
- 높은 환불·취소율 리스크
- 법적 규제 변화에 따른 리스크
- 청소년 보호 관련 법적 책임 우려

### 카드사 심사 제한
- 각 카드사가 자체 정책에 따라 특정 업종 결제 제한 가능
- 국민카드 등 일부 카드사는 **통신판매업 신고 미완료 시 심사 자체 거부**
- 담배 관련 업종은 카드사 내부 리스크 평가에서 **고위험**으로 분류

## 5. 배송 규제

### 국내 택배 배송
- 담배는 국내 택배 발송 시 **취급제한 품목**으로 분류되는 경우가 많음
- 우체국 EMS 기준: 담배 및 담배류(종류불문)는 **국제배송 금지품목**
- UPS: 전자담배 장치, 전자 액체, 젤을 포함한 베이핑 제품 **배송 거부**
- 국내 주요 택배사(CJ대한통운, 한진, 로젠 등)는 담배 제품에 대해 개별 정책 적용

### 본인확인 배송
- 담배사업법상 소매인 지정을 받은 자만 판매 가능 → **대면 판매 원칙**
- 온라인 판매 전면 금지 시행(2026.04.24) 후에는 택배 배송 자체가 법적으로 불가능해질 예정
- 현재도 일부 전자담배 쇼핑몰에서 배송지연 및 택배사 거부 문제 발생 중

### 해외 배송 규제
- 담배수입판매업 등록 필요 (2026년 4월 24일부터 합성니코틴 포함)
- 재정경제부 안전관리 기준 준수 의무
- 유해성분 검사 필수

## 6. SaaS 플랫폼(Cafe24/아임웹)에서 규제 구현이 어려운 이유

### 6.1 성인인증 시스템의 한계

| 문제 | 상세 |
|------|------|
| **기본 인증만 제공** | 카페24는 아이핀/휴대폰 인증 기반 성인인증을 제공하나, 담배 판매에 필요한 강화된 인증 프로세스 커스터마이징 불가 |
| **구매 시점 재인증 불가** | 회원가입 시 1회 인증 후 추가 인증 프로세스를 주문 플로우에 삽입하기 어려움 |
| **비회원 구매 통제 한계** | SaaS 플랫폼의 비회원 주문 프로세스에 성인인증을 강제 삽입하는 커스터마이징 불가 |
| **인증 유효기간 관리 불가** | 인증 후 일정 기간 경과 시 재인증을 요구하는 로직 구현 불가 |

### 6.2 상품 표시 규제 대응의 한계

| 문제 | 상세 |
|------|------|
| **법정 경고문구 자동화 불가** | 상품 등록 시 법정 경고문구·경고그림을 자동으로 삽입하는 기능 없음 |
| **성분정보 필드 부재** | 니코틴 함량, 유해성분 등 담배 전용 상품 속성 필드가 없음 |
| **상품 상세페이지 제약** | 아임웹은 위젯 기반으로 자체 코드/스크립트 커스터마이징이 거의 불가능 |

### 6.3 결제 연동의 구조적 한계

| 문제 | 상세 |
|------|------|
| **PG사 선택 제한** | 카페24/아임웹이 제공하는 PG사 옵션에서 전자담배 업종 승인이 거절됨 |
| **카페24페이먼츠 입점 불가** | 카페24 자체 결제 서비스에서도 담배 관련 업종은 입점 불가 업종으로 분류 |
| **결제 프로세스 커스텀 불가** | 결제 전 성인 재인증 단계 삽입, 특수 결제 조건 설정 등 불가 |
| **간편결제 사용 불가** | 네이버페이 등 주요 간편결제에서 전자담배 취급 금지 |

### 6.4 배송 연동의 한계

| 문제 | 상세 |
|------|------|
| **본인확인 배송 미지원** | 수령 시 본인확인을 강제하는 배송 옵션 설정 불가 |
| **택배사 API 커스텀 불가** | 특정 택배사와의 특수 계약(본인확인 배송 등) 연동이 SaaS 구조상 어려움 |
| **배송 제한 지역/조건 세분화 불가** | 담배 판매 가능 지역, 시간 등 조건부 배송 설정 제한적 |

### 6.5 광고/마케팅 도구의 규제 미준수

| 문제 | 상세 |
|------|------|
| **SNS 연동 자동 제한 불가** | 플랫폼의 SNS 자동 공유 기능이 담배 광고 규제를 자동으로 준수하지 않음 |
| **마케팅 이메일/푸시 제한 불가** | 비흡연자·청소년에게 마케팅 메시지가 발송되지 않도록 하는 필터링 부재 |
| **SEO/검색노출 제한 불가** | 검색엔진에서의 광고 노출을 법적 기준에 맞게 제한하는 기능 없음 |

### 6.6 종합: 왜 자사몰이 필요한가

SaaS 플랫폼의 근본적 한계는 **"범용 솔루션"**이라는 점에 있다:

1. **규제 특화 기능 부재**: 담배/전자담배는 주류, 성인용품과도 다른 고유한 규제 체계를 가지며, 범용 SaaS는 이를 반영한 전용 기능을 제공하지 않음
2. **코드 레벨 커스터마이징 불가**: 아임웹은 위젯 기반, 카페24도 제한된 스킨 수정만 가능하여 주문·결제·배송 프로세스의 근본적 변경이 불가
3. **외부 API 연동 제한**: 본인인증 API, 택배사 특수 배송 API, 법정 성분 DB 연동 등 외부 시스템과의 심층 연동이 어려움
4. **법 개정 대응 속도**: 2026년 4월 담배사업법 개정 등 빈번한 규제 변화에 SaaS 플랫폼이 즉시 대응하기 어려움
5. **PG 자유도**: 자사몰은 담배 업종 전문 PG사와 직접 계약이 가능하나, SaaS는 플랫폼이 제공하는 PG 옵션으로 제한
