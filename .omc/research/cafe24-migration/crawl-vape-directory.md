## 1. 네이버 쇼핑 "전자담배" 카테고리 — 입점 쇼핑몰 목록 확인 방법

### 접근 경로
| 단계 | URL / 행동 |
|------|-----------|
| 네이버 쇼핑 검색 | `https://shopping.naver.com/` → 검색창에 "전자담배 액상" 입력 |
| 가격비교 탭 | 검색 결과에서 **가격비교** 탭 클릭 → 해당 상품을 취급하는 쇼핑몰 목록이 노출됨 |
| 쇼핑 파트너센터 | `https://center.shopping.naver.com/` — 입점 쇼핑몰 관리/신청용 (판매자 측) |
| 네이버 쇼핑 카테고리 직접 접근 | `https://shopping.naver.com/` → 카테고리 탐색: 생활/건강 > 담배/전자담배 |

### 주의사항
- 네이버 쇼핑은 **CPC(클릭당 과금)** 기반으로 입점 쇼핑몰이 노출됨
- 전자담배는 성인인증 필수 카테고리로, 로그인 후 19세 이상 인증을 해야 검색 결과가 온전히 노출됨
- 네이버 쇼핑에 입점된 전자담배 쇼핑몰 **전체 목록**을 일괄 조회하는 공개 기능은 없음 → 상품 검색 → 가격비교에서 쇼핑몰별로 확인하는 것이 현실적 방법
- 네이버 스마트스토어에서 직접 전자담배 액상을 판매하는 셀러도 존재

## 5. 공정거래위원회 통신판매업 신고 사업자 조회

### 접근 경로

| 항목 | 내용 |
|------|------|
| 공정위 사업자 조회 | `https://www.ftc.go.kr/www/selectBizCommList.do?key=254` |
| 공공데이터포털 API | `https://www.data.go.kr/data/15126311/openapi.do` (통신판매사업자 등록현황 API) |

### 사용 방법
1. 공정위 사이트 접속 → "통신판매사업자" 메뉴
2. **상호명** 검색: "전자담배", "베이프", "vape", "액상", "전담" 등 키워드로 검색
3. **도메인** 검색: 알려진 전자담배 쇼핑몰 도메인을 입력하여 사업자 정보 확인
4. 조회 결과에서 확인 가능한 정보:
   - 신고번호
   - 사업자등록번호
   - 법인/개인 여부
   - 대표자명
   - 사업장 소재지
   - 도메인 (인터넷 주소)
   - 신고기관 (관할 지자체)

### 대량 조회 방법 (API 활용)
- 공공데이터포털의 **공정거래위원회_통신판매사업자 등록현황 API** 활용
- API URL: `https://www.data.go.kr/data/15126311/openapi.do`
- API 키 발급 후, 상호명에 "전자담배", "베이프", "vape" 등 키워드를 넣어 대량 조회 가능
- 주의: 상호명에 "전자담배"가 포함되지 않은 사업자는 누락됨 → 도메인 역조회와 병행 필요

### 한계
- 상호명/도메인 기반 검색이므로, 전자담배를 취급하지만 상호에 관련 키워드가 없는 사업자는 누락
- 업종별 분류 기능 없음 (전자담배만 필터링 불가)
- 간이과세자, 연 50회 미만 거래 사업자는 신고 의무 면제

## 6. Cafe24 마켓 — 전자담배 카테고리 조회 가능 여부

### Cafe24 기반 전자담배 쇼핑몰을 찾는 현실적 방법

| 방법 | 설명 |
|------|------|
| **도메인 패턴 검색** | `*.cafe24.com` 서브도메인 형태의 쇼핑몰 식별 (예: `vape9.cafe24.com`, `karivape1220.cafe24.com`) |
| **Wappalyzer 활용** | `https://wappalyzer.com/technologies/ecommerce/cafe24/` — Cafe24 플랫폼을 사용하는 사이트 목록 조회 (전체 목록이지만 키워드 필터 불가) |
| **BuiltWith 활용** | `https://builtwith.com/` — Cafe24 기술 스택 사용 사이트 검색 |
| **HTML 소스 확인** | 쇼핑몰 HTML에서 `cafe24` 키워드 존재 여부로 플랫폼 판별 (`img.echosting.cafe24.com`, `cafe24.com/lib/` 등) |
| **Exa 검색** | `site:cafe24.com` + "전자담배" 키워드로 서브도메인 수집 |
| **공정위 도메인 조회** | 통신판매업 신고 시 등록된 도메인에 `cafe24` 포함 여부로 필터링 |
