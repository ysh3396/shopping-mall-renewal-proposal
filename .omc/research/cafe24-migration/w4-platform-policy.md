## 1. Cafe24 전자담배 정책

### 1.1 상품 등록 가능 여부

- **Cafe24에서 전자담배 상품 등록 및 판매는 가능하다.**
- 실제 운영 사례가 확인됨:
  - `cigamall.cafe24.com` — 전자담배 도매 쇼핑몰 (회원 전용)
  - `vapedotcom.cafe24.com` — 전자담배 소매 쇼핑몰
- Cafe24 이용약관(제14조)에서 **담배/전자담배를 명시적 판매금지 품목으로 지정하지 않음**
- 약관은 "법령 위반 상품 판매 금지", "미풍양속 저해 금지" 등 **포괄적 기준**만 적용

### 1.2 카테고리 제한 정책

- Cafe24는 전자담배 전용 카테고리를 별도로 제공하지 않음
- 운영자가 자유롭게 카테고리를 생성하여 상품을 분류할 수 있음
- 성인인증이 필요한 상품은 개별 상품 또는 카테고리 단위로 **성인전용(19금) 설정** 가능

### 1.3 성인인증(연령인증) 기능

- **Cafe24는 자체 성인인증 기능을 기본 제공**
- 설정 위치: `상점관리 > 운영관리 > 운영방식설정 > 상점/보안설정`
- 본인인증사가 제공하는 **연나이 기준**으로 차단 여부 결정 (연나이 19세 이상 구매 가능)
- 상품별/카테고리별 개별 성인인증 설정 가능
- 성인인증 필요 쇼핑몰은 **인트로 화면**(첫 화면)을 별도 설정하여 상품이 바로 노출되지 않게 할 수 있음
- 카카오싱크 간편가입 후 별도 성인인증 연동 가능

**출처:**
- [쇼핑몰 성인인증 관련 자주 묻는 질문 – 카페24 Help Center](https://support.cafe24.com/hc/ko/articles/18810336169241)
- [회원 관련 설정 – 카페24 Help Center](https://support.cafe24.com/hc/ko/articles/7751401119257)

### 1.4 카페24페이먼츠(PG) 결제 연동

- 카페24페이먼츠에는 **입점 불가 업종 목록**이 별도 관리됨
- 공식 헬프센터 문서에 따르면, 입점 불가 업종은 "신청 페이지 또는 서비스 관리자 페이지"를 통해 별도 게재
- **담배/전자담배가 입점 불가 업종에 포함되는지 공식 확인 불가** (페이지 접근 제한)
- 단, 실제로 Cafe24 기반 전자담배 쇼핑몰이 운영 중이므로, **별도 PG사(NHN KCP, KG이니시스, 토스페이먼츠 등)를 통한 결제 연동은 가능한 것으로 추정**
- PG 심사 시 업종별 카드사 심사 정책에 따라 반려될 수 있으며, 추가 서류 제출이 필요할 수 있음

**출처:**
- [카페24페이먼츠 입점 불가 업종 – 카페24 Help Center](https://support.cafe24.com/hc/ko/articles/20160265659033)
- [PG신청, 심사, 서비스 관련 FAQ – 카페24 Help Center](https://support.cafe24.com/hc/ko/articles/17299990270105)

### 1.5 계정 정지/제한 사례

- Cafe24 자체에서 전자담배 판매를 이유로 계정을 정지한 공식 사례는 확인되지 않음
- 서비스 이용 제한 사유:
  - 이용약관 위반 게시글 반복 등록
  - 불법/부당 광고성 게시글 반복
  - 관련 기관으로부터 이용정지 조치 접수
- 법령 준수(담배사업법, 청소년보호법 등) 시 **운영 가능한 것으로 판단**

**출처:**
- [쇼핑몰 서비스 이용제한 사례 – 카페24 Help Center](https://support.cafe24.com/hc/ko/articles/8461564570009)

## 2. 아임웹(Imweb) 전자담배 정책

### 2.1 상품 등록 가능 여부

- **아임웹에서도 전자담배 상품 등록은 기술적으로 가능**
- 아임웹 이용약관(제9조)에서 **담배/전자담배를 명시적 판매금지 품목으로 지정하지 않음**
- 약관은 "관계법령에 위배되는 행위", "공서양속에 반하는 정보 게시" 등 포괄적 금지만 규정
- 단, **전자담배 전문 쇼핑몰 운영 사례는 검색에서 확인되지 않음** (Cafe24와 대비)

### 2.2 카테고리 제한 정책

- 아임웹은 자유로운 카테고리 생성 가능
- **특정 상품군에 대한 카테고리 제한 정책은 공식적으로 확인되지 않음**
- 카테고리별 회원 그룹 권한 설정 가능 (특정 회원 그룹에게만 상품 노출/판매)

**출처:**
- [상품 카테고리 권한 설정하기 – 아임웹](https://imweb.me/faq?mode=view&category=29&category2=40&idx=71437)
- [특정 사용자 그룹에게만 상품 판매하기 – 아임웹](https://imweb.me/faq?mode=view&category=29&category2=33&idx=71815)

### 2.3 성인인증(연령인증) 기능

- **아임웹은 자체 성인인증 기능을 제공하지 않으며, 외부 서비스 연동 필요**
- 외부 성인인증 서비스 예시: **비바톤(BBaton)** 무료 성인인증 서비스
- 비용 구조:
  - 초기 설치비: 30,000원 (부가세 별도)
  - 월 이용료: 30,000원 (부가세 별도)
  - 기본 제공: 월 666건
  - 초과 시: 1건당 45원
- 회원만 성인인증 가능하도록 설정하면 인증 비용 절감 가능
- 한 번 성인인증 완료한 회원은 재인증 불필요
- 만 14세 이상 가입 약관 설정 가능 (환경설정 > 약관 설정)

**출처:**
- [만 14세 이상 가입 약관 설정하기 – 아임웹](https://imweb.me/faq?mode=view&category=29&category2=47&idx=71745)
- [무료 성인인증 연동하기 – BBaton](https://www.bbaton.com/doc2/%EC%97%B0%EB%8F%99%ED%95%98%EA%B8%B0/)

### 2.4 PG 결제 연동

- 아임웹은 **NicePay** 등 외부 PG사 연동 지원
- PG 심사 실패 시 NicePay 사용 권장 (업종 제한이 상대적으로 적음)
- 전자담배 업종의 경우 **카드사 심사에서 반려 가능성 존재**
- PG 심사 통과를 위한 필수사항:
  - 1개 이상 상품 노출
  - 배송 정책 명시
  - 비회원 구매 가능 설정
  - 하단 필수 정보 노출 (사업자등록번호, 통신판매업 신고번호 등)

**출처:**
- [PG 심사 통과를 위한 필수 확인사항 – 아임웹](https://imweb.me/faq?mode=view&category=29&category2=39&idx=71926)
- [국내 전자결제(PG) 신청 및 설정하기 – 아임웹](https://imweb.me/faq?mode=view&category=29&category2=39&idx=71419)

### 2.5 계정 정지/제한 사례

- 아임웹에서 전자담배 판매를 이유로 계정을 정지한 공식 사례는 확인되지 않음
- 이용약관 제8조에 따른 서비스 제한 사유:
  - 등록 상품이 불법정보에 해당하여 방송통신심의위원회 등에서 행정조치 접수 시
  - 사업자정보 허위 기재 또는 사기 쇼핑몰로 판단될 경우
  - 홈페이지를 범죄 도구로 사용한다고 판단될 경우

## 4. 결제 연동 제한 (공통)

### PG사/카드사 심사 정책

- 전자담배는 **업종 특성상 PG사/카드사 심사에서 제한될 가능성이 높음**
- 신용카드 이용불가 업종에 해당할 경우 PG 신청 자체가 불가
- 각 PG사별 업종 제한이 다르며, **NicePay가 상대적으로 업종 제한이 적음**
- 심사 반려 시 쇼핑몰 수정 + 추가 서류 제출 후 재심사 (14일 이상 소요 가능)
- 성인용품 업종과 유사하게, 전자담배도 **별도 심사 프로세스**를 거칠 가능성이 큼

### 필요 서류 (일반 PG 심사)
- 사업자등록증
- 통신판매업 신고증
- 대표자 신분증
- 담배소매업/수입판매업 허가증 (전자담배 관련 추가 서류)

## 6. 핵심 시사점

1. **Cafe24가 전자담배 쇼핑몰 운영에 더 적합**: 성인인증 기본 내장, 실제 운영 사례 다수 존재
2. **아임웹은 성인인증 추가 비용 발생**: 외부 서비스 연동 필요, 월 고정비 + 건당 비용
3. **두 플랫폼 모두 전자담배를 명시적으로 금지하지 않음**: 법령 준수가 핵심 조건
4. **2026년 4월 카카오 정책 변경 주의**: 카카오 로그인/결제 연동 시 성인인증 절차 필수화
5. **PG 심사가 최대 난관**: 전자담배 업종 특성상 카드사 심사 반려 가능성 높으며, 별도 서류 및 재심사 과정 필요
6. **담배사업법 개정 (2026.04.24 시행)**: 합성니코틴 규제 강화로 담배수입판매업 등록 등 추가 법적 요건 충족 필요
