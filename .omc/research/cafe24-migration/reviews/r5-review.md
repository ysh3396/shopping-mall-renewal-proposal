## 2. 강점

- **구조적 체계성**: 성인인증 → 결제 → 디자인 → API → 성능 → 데이터 → 운영자 후기 → 종합 리스크의 흐름이 논리적이며 전자담배 업종 특수성과 일반 SaaS 한계를 모두 다룸.
- **종합 리스크 테이블(§8)**: 항목별 위험도와 설명을 한눈에 볼 수 있어 의사결정자에게 즉시 유용. "치명적/높음/중간" 3단계 구분이 명확.
- **숨겨진 비용 정량화(§7)**: 최소 40~50만원 초기 비용, PG 가입비 22만원, 리뷰톡톡 월 119,000원 등 구체적 수치가 실무 가치를 높임.
- **데이터 이동성 섹션(§6)**: 이전 가능/불가능 데이터를 명확히 구분하고 SNS 연동 회원 데이터 Lock-in 문제를 실질적 위험으로 제시한 점이 유용.
- **실제 운영자 인용(§5)**: XETOWN 커뮤니티의 복수 사용자 발언을 통해 공유 호스팅 속도 문제를 생생하게 뒷받침함.

## 3. 약점

### MAJOR — 카카오싱크 성인인증 주장이 병행 리서치와 정반대로 충돌

w5 §1은 `"카카오싱크(간편 로그인) 사용 시 성인인증을 별도로 진행할 수 없는 구조적 문제 발생"`이라고 서술하며, 근거로 아이보스 커뮤니티의 운영자 **질문글**(답변 미확인)을 인용한다.

그러나 동일 프로젝트의 w4-platform-policy.md §1.3은 카페24 공식 Help Center를 직접 출처로 하여 `"카카오싱크 간편가입 후 별도 성인인증 연동 가능"`이라고 명시한다.

질문글은 답변이 확인된 출처가 아니다. 공식 Help Center와 충돌하는 주장에 커뮤니티 질문글을 근거로 사용하는 것은 신뢰도 기준 미달이다. 이 주장은 공식 문서로 재검증하거나 삭제해야 한다.

### MAJOR — PG "치명적 장벽" 단정이 반대 증거를 누락

w5 §2는 PG 결제 연동을 `"전자담배 업종 치명적 장벽"`으로 규정한다. 그러나 w4는 `cigamall.cafe24.com`, `vapedotcom.cafe24.com` 등 **실제 운영 중인 Cafe24 기반 전자담배 쇼핑몰**을 확인했다. 이는 PG 연동이 불가능하지 않으며, 특정 PG사(NicePay 등)를 통해 통과 가능함을 시사한다.

"치명적 장벽"은 아예 불가능한 경우에 쓸 수 있는 표현이다. 실제로는 "어려움이 있으며 PG사 선택이 중요"가 정확한 서술이다. 과장된 단정은 문서 전체의 신뢰도를 훼손한다.

### MAJOR — 합성니코틴 온라인 판매 전면 금지(2026.04.24) 맥락 완전 누락

w9-community-voices.md 및 전체 프로젝트 컨텍스트에서 확인된 바, **2026년 4월 24일부터 합성니코틴 액상형 전자담배의 온라인 판매가 전면 금지**된다. 이 규제는 "Cafe24 플랫폼의 한계" 전체를 논하는 이 문서의 가장 근본적인 배경이다.

결론(§9)이 `"자체 구축(독립몰)이 전자담배 업종에서는 장기적으로 더 적합"`이라고 권고하지만, 합성니코틴 온라인 판매 금지 아래에서는 독립몰 구축 자체가 의미를 상실할 수 있다. 이 맥락 없이는 결론의 유효성이 근본적으로 훼손된다.

### MINOR — 데이터 이동성 섹션의 맥락 혼동

§6의 `"카페24 간 이전 불가"` 항목은 Cafe24 내부 쇼핑몰 간 복사/이전 불가를 의미하지만, 섹션 맥락은 외부 플랫폼으로의 이전 문제를 다루고 있다. 독자가 "Cafe24에서 다른 플랫폼으로의 이전이 불가"로 오독할 수 있으며, 이는 사실과 다르다.

### MINOR — API Rate Limit 서술의 모호성

§4의 `"최대 10개/스토어"` 서술이 동시 호출 버킷 최대 크기(concurrency limit)인지 총 허용 호출 횟수인지 불명확하다. 버킷 방식(초당 1개씩 감소)의 메커니즘은 맞으나, 실제로는 스토어당 초당 허용량과 버킷 크기가 별도로 존재할 수 있다. 공식 Developers 문서(출처 인용됨)의 정확한 수치를 그대로 인용하거나, "동시 처리 가능한 최대 요청 수"로 표현을 명확히 해야 한다.

### MINOR — 코드엠샵 출처의 범용성 문제

PG 불가 업종 인용 출처가 카페24 공식 문서가 아닌 코드엠샵(카페24 전문 에이전시)의 가이드 페이지다. 이 정보는 특정 PG사 기준이며 카페24 환경 전체에 일반화하기 어렵다. 카페24페이먼츠 공식 입점 불가 업종 페이지(w4에서 확인된 링크)를 직접 인용했어야 한다.

## 4. 핵심 인사이트

1. **문서가 설득 자료로 작성되었다**: 자체 구축(독립몰)이라는 결론을 향해 역방향으로 증거를 선택하는 패턴이 보인다. 반대 증거(실제 운영 사례, w4의 공식 문서 기반 서술)가 체계적으로 누락되어 있다.

2. **규제 맥락의 부재가 문서의 결론을 무력화한다**: 카페24 플랫폼 한계를 논하기 이전에, 해당 업종 자체의 온라인 판매 가능성이 법적으로 종료되는 시점이 4주 앞으로 다가와 있다. 이 맥락 없이 "독립몰이 적합하다"는 결론은 공허하다.

3. **성인인증 섹션이 가장 실용적이다**: §1의 PC/모바일 분리 문제, 카카오싱크 연동 복잡성(논란이 있더라도 이슈 자체는 존재함), 팝업 UI 커스터마이징 불가는 실제 운영 시 체감되는 구체적인 한계이며, 플랫폼 전환 논거로서 가장 견고하다.

4. **데이터 Lock-in 섹션(§6)이 두 번째로 견고하다**: SNS 연동 회원 데이터, 적립금 히스토리, 리뷰 데이터의 이전 불가는 공식 확인이 가능하며 경쟁 우위 논거로 직접 활용 가능하다.
